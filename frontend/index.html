Local File Sharing Service

Welcome to the File Sharing Service

This simple service allows you to:

  • Upload files of up to 50MB
  • Get a unique 6-character code link for each file
  • Share the link with others on your local network
  • Download files using the provided link

Ready to share a file?

Go to Upload Page

How It Works

  1. Click the button above to go to the upload page
  2. Select a file from your computer
  3. Click "Upload File"
  4. After uploading, you'll receive a unique link
  5. Share this link with anyone who needs to download the file
  6. The recipient can access the file by visiting the link

Note: This service runs only on your local network. Files are accessible only to people connected to the same network.
